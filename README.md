# Ugly Finance Tool - mobile batch 1

- Mobile bottom nav: bigger icons, the "More" button is now a vertical (⋮) with no
  word label, and the whole bar sits higher so it isn't clipped by the screen edge.
- Home store cards now open that store's SALES DETAIL (Insights, daily view). A
  small "+ Enter today's numbers" button on each card still goes to entry.
- Top-right name + Sign Out are now a single account menu (avatar + ⋮). It holds the
  name/role, a Notifications link (with pending badge), an enable-push action, and
  Sign Out.
- Expense entry: editable date, and Card/Cash/Transfer method (see 19_expense_method.sql).
- Cash calendar: red dot on days with cash expenses, showing cash sales − cash spent
  = expected envelope.
