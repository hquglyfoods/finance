VAPID 개인키 변환기

개인키(PEM)를 붙여넣기 사고가 안 나는 한 줄(base64)로 바꿔줍니다.
🔒 이 페이지는 인터넷에 아무것도 보내지 않아요. 변환은 전부 이 브라우저 안에서만 일어나서 키가 밖으로 나가지 않습니다. 안심하고 쓰셔도 돼요.

1개인키(PEM)를 여기에 붙여넣으세요

Netlify → Site settings → Environment variables → VAPID_PRIVATE_KEY 값 전체.
-----BEGIN PRIVATE KEY----- 부터 -----END PRIVATE KEY----- 까지 다 복사해서 붙여넣으세요.

2이 한 줄을 복사하세요

3Netlify에 붙여넣기

  1. Netlify → Site settings → Environment variables
  2. VAPID_PRIVATE_KEY 를 찾아 값을 방금 복사한 한 줄로 교체
  3. 저장 (Save)
  4. Deploys 탭 → Trigger deploy (환경변수는 새로 배포해야 적용돼요)
  5. 앱 → Install App & Notifications → Diagnostics → Send test push 로 확인
